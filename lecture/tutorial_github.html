Tutorial on Setting up Github access on your own computer

  1. log into your github account

  2. Navigate to my Github page, for example, https://github.com/lewangecon/stat22 (or whatever the course page link is).

  3. Click Fork

  4. Click Code and copy and paste the HTTPS url.

  1. Rstudio, click File > New Project,

  3. Click Git

  4. Copy and paste the HTTPS url in the prompt Repository URL

  5. Create a new project, and Done!
